# Blockchain Scanner

Real-time blockchain analytics tool for monitoring transactions, whale movements, and DeFi protocols.

## Features
- Multi-chain support (ETH, BSC, Solana, Polygon)
- Whale wallet tracking
- Token price alerts
- DeFi protocol risk scoring
- Telegram/Discord notifications

## Installation
```
pip install -r requirements.txt
```

## Usage
```
python main.py scan --chain eth --wallet 0x...
python main.py alerts --token USDC
```
